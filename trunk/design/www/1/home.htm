What are zenFolders ?

zenFolders are virtual folders that display files returned as search results from Google Desktop Search. These folders do not really exist on your computer. You can not put files into them. They only appear as real folders in Windows Explorer, but their content is dynamicaly generated from search results.

Why are zenFolders useful ?

Google Desktop Search itself is a very useful tool. It helps us find information stored anywhere on our computers or even across the local network without us having to know where that information is physicaly located. However, it presents this information in the browser which is not always so convenient. We cannot copy files from the browser or drag them to another application and file properties dialog is available only in Internet Explorer. And, most important, we can not save those searches so we always have to type the full search string. zenFolders allow us to save those searches and group them in a way that makes sense to us. This way, we can have files appearing in several folders. In addition to that, we can create special folders that contain only files that have been created or accessed in the specified period, ie. last 24 hours. Google desktop Search string for this is: "filetype:htm datefrom:yesterday dateto:today", assuming that we want to see all htm files created or accessed since yesterday.

What are the requirements for using zenFolders ?

zenFolders require Google Desktop Search to operate. Google Desktop Search is a part of Google Pack which can be downloaded here ( after I obtain an AdSense account, I will put a different link). zenFolders have been tested on Windows2000 and Windows XP, however, Google Pack is available only on WindowsXP. For Windows2000 you must download Google Desktop.
