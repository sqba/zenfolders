The idea of saving searches and representing the results as regular folders is not new. It seems to originate from 1996 when BeOS introduced it under the name "Tracker". In 2005, when apple released the new version of Mac OS X v10.4 named "Tiger", it contained a feature called "Smart Folders". Then, in 2006, a multi-platform free desktop environment Gnome reached version 2.14, and it allowed searches to be saved into called "Saved Searches". And, eventually, in January 2007, Microsoft releases it's new operating system called Vista which includes the "Saved Searches" feature.
