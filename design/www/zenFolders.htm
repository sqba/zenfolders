Your files. When and where you need them.

What are zenFolders ?

zenFolders are 'virtual' folders, that is, folders that do not really exist on your computer. You can not put files into them. They only appear as real folders in Windows Explorer, but their content is dynamicaly generated from search results.

Why are zenFolders useful ?

Today's hard drive capacities allows us to store enormous ammounts of information on them, and so we do. This information just piles up and classifying it requires an effort. zenFolders help us classify that information easely, whenever we want.

Can you give me an example when zenFolders are useful ?

Think about it: you are working on a paper about Nikola Tesla. You've collected all sorts of files about the great inventor on your PC, but you lost track of all the places where these reside. So, you navigate your Windows Explorer to zenFolders, create a new folder, name it "Nikola Tesla", and the folder is instantly filled with all sorts of material that mentions Tesla. But, there are all sorts of files inside which makes browsing a bit harder. So, we probably want to get rid of '.css' files, since most of the material are web pages, and css files are usually linked to html files. No problem, just right click on any ".css" file in the list, select "Hide all 'css' files" from the popup menu, and you won't be seeing any ".css" files in this folder. Also, since you have a lot of ".pdf" files, you would like to have them grouped somehow. Again, right click on any ".pdf" file and select option "Move all .pdf files to 'pdf files' folder". Now you have a subfolder with all pdf documents that mention Nikola Tesla. How convenient is that?

Ok, but I am not working on Tesla, how can I use zenFolders ?

Probably one of most convenient uses for zenFolders are folders with recently accessed files. If you are like me, every day you save a couple of articles from the internet for later reading. Usualy they have different topics, but you don't want to classify them into folders when you save them. Instead, create a zenFolder with html files that have been accessed or created since yesterday. Google desktop search string for this is: "filetype:htm datefrom:yesterday dateto:today". And now you have a folder where you can find all those htmls that you saved today, or day before. Also, using same search commands, you can create folders for specific dates.
